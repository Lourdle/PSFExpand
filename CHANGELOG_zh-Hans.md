# PSFExpand 与 PSFExtractionHandler 更新日志

## PSFExpand
- 2.0.1.0
1. 适配 API 版本 2.0.1，支持设置文件时间。
2. 弃用通过调用 Expand.exe 展开 CAB 文件，转用 PSF Extraction Handler Cabinet 实用函数，提升速度。
3. 修正将文件展开到根目录会出错的问题。
4. 优化进度报告机制，提升速度。
5. 弃用“SafeRead”选项。
6. 使用 Shlwapi 匹配文件通配符。
7. 新增启动时检测 API 版本是否兼容的功能。
8. 其他未列出的细节修正。

- 1.1.1.0
1. 升级默认 API 版本为 1.1.1。
2. 新增“SafeRead”选项。
3. 修正几处文档错误。
4. 其他未列出的细节修正。

- 1.0.2.0
1. 修复选择 PSF 文件时文件名错误的问题。
2. 新增文件展开前显示相关文件的功能。
3. 优化文件处理机制，支持长文件名。
4. 升级默认 API 版本为 1.0.2。
5. 其他未列出的细节修正。

- 1.0.1.1
1. 修复筛选器处理 '?' 失败的问题。
2. 新增显示语言主动选择功能。
3. 新增 CAB 文件扩展与自动选择 PSF 文件和 XML 文件功能。
4. 其他未列出的细节修正。


- 1.0.1.0
首发版本。


## PSFExtractionHandler
- 2.0.1
1. 优化文件处理机制。
2. 新增 Cabinet 实用函数。
3. 弃用多线程读取 XML。
4. 新增文件时间设置支持。
5. 函数在接受的不支持的标识会自动忽略而不是失败。
6. 多线程并发处理对象支持。
7. 优化线程安全。
8. 其他未列出的细节修正。

注意：从此版本开始将不再兼容早期的 API 版本。未经适配的程序调用此版本以后的 API 的部分函数，其行为是未定义的。

- 1.1.1
1. 取消文件直写，提升文件写入速度。
2. 修复单线程不展开整个文件的问题。
3. 支持以单线程读取 XML 文件，以避免因小概率冲突异常导致的程序失败。
4. 其他未列出的细节修正。

- 1.0.2
1. 优化文件处理机制，支持长文件名。
2. 新增 PSF 文件格式有效性检查。
3. 优化并行多线程调度。
4. 其他未列出的细节修正。

- 1.0.1
首发版本。
